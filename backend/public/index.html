How It Works

Three steps. That's it.

No setup wizardry. No learning curve. Just markdown.

Jot it down. Move on.

Add tasks to tasks.md or your daily file. No menus, no dropdowns — just type. Dates and projects are optional metadata, not mandatory form fields.

  • Quick-capture in tasks.md with optional due dates
  • Scratchpad in inbox.md for ideas and notes
  • Add directly to today's daily file 
# tasks.md

- [ ] Write project proposal | 2026-02-20 | work
- [ ] Buy groceries | 2026-02-18
- [ ] Research hosting options

It organizes itself.

Tasks auto-route to the right project folder. Every morning, a daily file is generated with everything that's due — pulled from all your projects automatically.

  • AI matches tasks to projects (or you can specify)
  • Daily file assembles itself each morning
  • Projects have goals, milestones, and monthly task files 
# Monday, February 16, 2026

## Overdue
- [ ] Finish budget report (finance)

## Today
- [ ] Write project proposal (work)
- [ ] Buy groceries (personal)

Check it off. Everything syncs.

Work from your daily file. Check off a task and it's marked done in the project file automatically. No updating in two places. No drift.

  • Check off tasks in the daily file — syncs everywhere
  • Dashboard shows progress across all projects
  • AI daily briefings summarize your workload

## Today
- [x] Write project proposal (work)
- [x] Buy groceries (personal)

Features

Built for how you actually work.

No bloat. No learning curve. Just the features you need.

Plain Markdown

Your data is .md files. Open them in any editor, sync with any service, own them forever.

Daily Auto-List

Every morning, a focused to-do list is generated from all your projects. No curation needed.

Project Tracking

Each project has goals, milestones, and monthly task files. Strategic and tactical views, always in sync.

Optional AI

Smart search, auto project inference, personalized daily briefings. Bring your own API key — or skip it entirely.

Built-in IDE

VS Code-style editor in your browser. File tree, Monaco editor, dashboard — all in one window.

100% Local

No servers, no databases, no sign-up. Runs on your machine. Your files stay on your hard drive.
